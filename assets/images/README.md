## Optimization

After placing images here, optimize them using:
- TinyPNG: https://tinypng.com/
- Squoosh: https://squoosh.app/
- ImageOptim (Mac)

Target file sizes:
- Hero/Product images: < 200KB each
- Thumbnails: < 50KB each
- Logos: < 30KB each
